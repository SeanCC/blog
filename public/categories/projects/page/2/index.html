Random Walkers and Web Crawlers

by Sean Carter
Tags: projects quickie web crawling random walks random graphs
Categories: Projects
Brief Overview: This was put together for a class assignment in Probability Theory 2. We were to discuss random walks and how they relate to the PageRank algorithm. An overview of the relevant mathematics is given, and a small web crawler is implemented to collect links between wikipedia articles. The links are saved in a SQLite database and used later to construct a graph and rank articles based on centrality.
